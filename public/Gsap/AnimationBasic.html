HTML Animation Basics

This page demonstrates **core animation techniques in HTML**, including CSS Transitions, CSS Keyframes, and JavaScript animations.

1. CSS Transition

Transitions allow smooth changes of CSS properties when an element changes state (like hover).

Hover Me
Code Example:
.transition-box {
  transition: transform 0.5s ease, background-color 0.5s ease;
}
.transition-box:hover {
  transform: scale(1.3) rotate(15deg);
  background-color: #facc15;
}
Explanation: Hovering over the box smoothly scales and rotates it while changing color.

2. CSS Keyframes Animation

Keyframes allow defining multiple steps over time for more complex animations.

Animate
Code Example:
@keyframes slideFade {
  0% { transform: translateX(0); opacity:0; background-color:#38bdf8; }
  50% { transform: translateX(50px); opacity:0.5; background-color:#f87171; }
  100% { transform: translateX(100px); opacity:1; background-color:#34d399; }
}
.keyframe-box {
  animation: slideFade 2s ease forwards;
}
Explanation: The box moves, fades in, and changes color over 2 seconds.

3. JavaScript Animation

JS animations allow full dynamic control over properties, timing, and interaction.

JS
Code Example:
const box = document.getElementById('jsBox');
let pos = 0;
const interval = setInterval(() => {
  if(pos >= 100){ clearInterval(interval); return; }
  pos += 2;
  box.style.transform = `translateX(${pos}px)`;
  box.style.opacity = pos/100;
}, 20);
Explanation: JS changes CSS properties over time using `setInterval`. You can add interactivity, loops, or triggers.

4. Summary of HTML Animation Methods

  • CSS Transitions: Best for simple hover/focus effects, lightweight, easy to use.
  • CSS Keyframes: More complex, timeline-based animations. Good for repeating, looping, or sequential steps.
  • JavaScript Animations: Most flexible. Can animate dynamically, respond to user input, and control sequences.
Tip: For professional interactive websites, JS libraries like GSAP or Anime.js provide advanced animation tools.
