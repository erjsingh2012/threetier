6. Semantic Color Variables

Use descriptive names instead of raw hex codes throughout your app:

  • Primary: Main branding color
  • Secondary: Highlights or call-outs
  • Accent: Buttons or icons
  • Background: Light or dark surfaces
  • Text: Foreground readability

7. Light/Dark Theme Switch (Concept)

/* Default: Light */
:root {
  --background: white;
  --text: black;
}

/* Dark Theme */
body.dark-mode {
  --background: black;
  --text: white;
}

This allows toggling themes using class="dark-mode".
