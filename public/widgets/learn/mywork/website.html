CSS Essentials Cheat Sheet

Learn the 80% most-used CSS concepts with examples and best practices.

1. Selectors & Specificity

Use classes (.box), IDs (#main), and element selectors wisely.

p { color: black; }
.highlight { color: red; }

2. Box Model

Every element is a box: content + padding + border + margin

box-sizing: border-box;

3. Flexbox (1D Layout)

Use for rows or columns. Automatically handles spacing and alignment.

A
B
C
display: flex; justify-content: space-between; align-items: center;

4. Grid (2D Layout)

Create columns and rows easily.

1
2
3
display: grid; grid-template-columns: repeat(3, 1fr); gap: 10px;

5. Positioning

Use static, relative, absolute, fixed, and sticky to control element position.

position: absolute; top: 10px; left: 10px;

6. Typography

Control font-family, size, line-height, and weight.

font-size: 16px; line-height: 1.5; font-weight: bold;

7. Colors & Backgrounds

Use named colors, HEX, RGB, or HSL.

background-color: #f0f0f0; color: #333;

8. Spacing: Margin, Padding, Border

These control element spacing and separation.

margin: 10px; padding: 15px; border: 1px solid #ccc;

9. Display & Visibility

Control if elements are shown, hidden, or inline/block.

display: none; visibility: hidden; display: inline-block;

10. Responsive Design (Media Queries)

Change layout based on screen size.

@media (max-width: 600px) { ... }

11. Transitions & Hover Effects

Animate changes smoothly.

transition: all 0.3s ease;
.box:hover { background: #2b6cb0; }
