  1. Basic 2D Grid: Fixed rows/columns. Ideal for dashboards, pricing tables.

  8. Auto-placed Grid: Items flow without specifying placement.
  9. Complex Grid with Span: Highlighted feature blocks using grid-column.
  10. Nested Grid: A grid within a grid for component layout.
  11. Centered Grid: Center content both vertically and horizontally.
  12. Full-Width Hero Grid: Edge-to-edge banner layout.
  13. Grid with Gapless Rows: No spacing between grid items.
  14. Equal Height Cards Grid: Responsive cards with uniform height.
  15. Overlay Text Grid: Image cards with text positioned over them.
