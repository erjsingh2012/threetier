1. font-family

Specifies the typeface of text. Always include a fallback.

This is Arial (sans-serif)
This is Georgia (serif)
This is Courier New (monospace)
This is Times New Roman (serif)

2. font-size

Controls the size of text using units like px, em, rem, %, etc.

12px font-size
1em font-size (relative to parent)
1.5rem font-size (relative to root)
120% font-size

3. font-weight

Controls thickness of text. Accepts keywords or numbers.

font-weight: 100 (Thin)
font-weight: 400 (Normal)
font-weight: bold
font-weight: 900 (Heavy)

4. font-style

Specifies italic or normal.

font-style: normal
font-style: italic
font-style: oblique

5. font-variant

Used to show small-caps versions of text.

font-variant: normal
font-variant: small-caps

6. line-height

Adjusts the vertical space between lines of text.

line-height: 1
line-height: 1.5 (Common)
line-height: 2

7. letter-spacing

Adjusts space between characters.

letter-spacing: normal
letter-spacing: 2px
letter-spacing: -1px

8. word-spacing

Adjusts space between words.

word-spacing: normal spacing
word-spacing: 10px

9. text-transform

Controls capitalization of text.

text-transform: none
text-transform: uppercase
TEXT-TRANSFORM: LOWERCASE
text transform capitalize

10. font (shorthand)

Shorthand to define all font properties in one line.

italic small-caps bold 20px Georgia

11. Font Stack Example

Good practice: use multiple font fallbacks.

12. Responsive Font Units

Use vw and em for responsive scaling.

font-size: 2vw (scales with screen width)

13. Using Google Fonts

Import fonts via Google Fonts using <link> in HTML head.

<link href="https://fonts.googleapis.com/css2?family=Roboto&display=swap" rel="stylesheet">
