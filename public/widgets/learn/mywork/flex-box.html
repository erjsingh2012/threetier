What is Flexbox?

Flexbox is a one-dimensional layout system in CSS that makes it easy to align and distribute space among items in a container, even when their size is unknown or dynamic.

Display: flex

We start by turning a container into a flex container using:

.container { display: flex; }

Flex Direction

The flex-direction property defines the direction of the main axis. You can use row, row-reverse, column, or column-reverse.

.container { display: flex; flex-direction: row | column; }

Justify Content

This aligns items along the main axis (horizontally for flex-direction: row).

.container { justify-content: flex-start | center | space-between | space-around | space-evenly; }

Align Items

This aligns items along the cross axis (perpendicular to the main axis).

.container { align-items: flex-start | center | flex-end | stretch | baseline; }
